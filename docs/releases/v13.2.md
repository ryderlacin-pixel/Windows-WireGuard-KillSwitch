# v13.2 — No Destructive Rescue

Removes `kurtar.bat`, `kurtar.ps1`, `kurtar2.ps1`, and `resume-after-unbrick.ps1`. These scripts disabled tasks, stopped services, and deleted all firewall rules — which felt like uninstalling the kill switch.

## Replacement

- **Watchdog** `Invoke-GentleUnbrick` — removes outbound blocks + DNS leak rules only.
- **Watchdog** `Invoke-DeepUnbrick` (streak ≥ 5) — gentle unbrick + `UnbrickUntil` 10 min + `BootGrace` 180s. Tasks and `WGKillSwitchSvc` stay running.
- **Monitor** `Invoke-EmergencyUnbrick` — same deep gentle logic inline; never spawns external teardown scripts.

## If internet is stuck

1. Wait 1–5 min for `WG-InternetWatchdog`.
2. Re-run `install.ps1` as Administrator if needed.
