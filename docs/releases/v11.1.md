# v11.1 — Monitor singleton fix

## What's new

| Area | Change |
|------|--------|
| **Single launcher** | Install STEP 19 and `repair.ps1` no longer double-start monitor (`schtasks` + direct) |
| **Dedupe** | `Remove-OtherMonitorProcs` at startup + every ~30s in monitor loop |
| **PID file** | Stale `monitor.pid` cleaned when process is dead |
| **Mutex** | 5s acquire wait (was 0ms) — fewer race duplicates |
| **Tamper** | Monitor re-applies `Enable-Block` every cycle while unhealthy |
