Builds on v15.0 strong privacy stack. No change to protection behavior when already on v15.0 — this release improves maintainability, documentation honesty, and Hassas-Tarama UX.

## What changed

- Orchestrator [`install.ps1`](../install.ps1) (~80 lines) dot-sources 8 modules under [`lib/`](../lib/)

- Easier code review; test-suite reads heredocs from `lib/Install-GeneratedScripts.ps1`

### Documentation (WARP-first)

- README and CODE_REVIEW aligned with v15.0 release: **free WARP default**, Tor for sensitive sessions, paid VPN optional

### Dev hygiene

- `.gitignore` covers bisect/emergency scripts and local install logs
- CI parses `lib/*.ps1` and validates `live-smoke-test.ps1` structure offline

### Optional CI live smoke

- [`scripts/live-smoke-test.ps1`](../scripts/live-smoke-test.ps1) — read-only; **SKIP** when stack not present (safe on GitHub runners)
