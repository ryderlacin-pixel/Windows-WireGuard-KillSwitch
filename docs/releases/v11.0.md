# v11.0 — Ultimate hardening (production release)

## What's new

| Area | Change |
|------|--------|
| **Self-repair** | `Repair-ConfigIntegrity` fixes BOM/IPv6 in config; `Repair-EssentialFirewall` restores DNS/WARP rules |
| **Network change** | `Test-NetworkChanged` + `NetworkFingerprint` registry — modem/route change triggers DNS flush + delayed-auto |
| **Boot resilience** | `Ensure-DelayedAutoStart` + `Test-DelayedAutoStart` (`sc qc` DELAYED); GPO waits up to 120s for healthy stack |
| **WMI stability** | 90s cooldown (`wmi-cooldown.txt`) prevents repair storms |
| **Faster recovery** | `WG-RepairTask` every **2 min** (was 5); service-monitor tunnel-down poll **15s** |
| **Stress gate** | `scripts/ultimate-stress-test.ps1` — live kill/tamper/network/reinstall scenarios |

## Unchanged from v10.9

- Install-safe mode (`install.inprogress` + `kurtar.bat`)
- IPv6 strip, WMI subscription, monitor singleton, 2-of-3 `Test-Internet`
- Security audit 32-check suite
