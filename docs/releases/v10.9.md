# v10.9 — Security hardening (public release)

## Fixes (audit WARN → PASS)

| Issue | Fix |
|-------|-----|
| `::/0` in WARP config | `Remove-IPv6FromConfig` strips IPv6 from Address/DNS/AllowedIPs |
| IPv6 adapter bindings | `Disable-AllIPv6Bindings` on every present adapter |
| Duplicate monitor (2 PIDs) | `Stop-AllMonitorProcs` before activate; repair mutex gate; 8s task wait |
| WMI subscription missing | `Install-WmiSubscription` with simplified WQL (no CommandLine LIKE) |
| Slow kill switch on tunnel drop | 2s monitor poll; immediate block on tunnel loss; service 30s tunnel poll; 60s recovery hold |
