# v11.2 — Post-reboot auto-verification (current production)

## What's new

| Area | Change |
|------|--------|
| **WG-RebootVerify** | Scheduled task: 5 min after boot, runs full health audit |
| **post-reboot-verify.ps1** | Runs `post-install-verify.ps1` + `security-audit.ps1` |
| **Logging** | `C:\WireGuard\reboot-verify.log` + registry `RebootVerifyLastResult` |
| **Once per boot** | Marker file `reboot-verify-<bootkey>.done` prevents duplicate runs |

## Unchanged from v11.1

- Monitor singleton, self-repair, ultimate stress gate, 32-check security audit

## Verify

```powershell
.\install.ps1
# Reboot PC — wait ~5 min, then:
Get-Content C:\WireGuard\reboot-verify.log -Tail 20
Get-ItemProperty HKLM:\SOFTWARE\WGKillSwitch RebootVerifyLastResult
```

Expected: `RebootVerifyLastResult = PASS`, log ends with `VERDICT: POST-REBOOT PASS`.

Manual run:

```powershell
.\scripts\post-reboot-verify.ps1 -Force
```
